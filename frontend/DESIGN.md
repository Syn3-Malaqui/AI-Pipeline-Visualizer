# Architectural Precision Layer

## Token mapping
- `background`: workspace canvas
- `surface-container-low`: secondary panel surfaces
- `surface-container-high`: primary panel surfaces
- `surface-container-highest`: input cards and node cards
- `primary`: primary actions
- `on-surface`: default text

## Interaction guidance
- Prefer tonal separation (`surface-container-*`) instead of hard section borders.
- Use compact technical tags (`rounded-sm` style equivalent) for metadata chips.
- Keep the right pipeline pane information dense and visually calm.
